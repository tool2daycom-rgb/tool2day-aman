# أمان · aman.tool2day.com

منصة عربية تفاعلية من Tool2Day توجّه ضحايا الابتزاز والانتهاك الرقمي خطوة بخطوة — بدون طلب صور أو تخزين بيانات شخصية.

## المسارات

- `/` الصفحة الرئيسية
- `/guide` التوجيه التفاعلي (سؤال واحد في كل شاشة)
- `/stopncii` شرح StopNCII بالعربي (بالغون)
- `/take-it-down` شرح Take It Down بالعربي (قاصرون)
- `/safe-browsing` نصائح التصفح الآمن
- `/privacy` الخصوصية

## مبادئ الخصوصية

- لا رفع صور
- لا تسجيل حساب
- لا Google Analytics
- اختيارات المسار فقط في `sessionStorage` على جهاز المستخدم
- زر خروج سريع + Escape

## التشغيل محلياً

```bash
npm install
npm run dev
```
